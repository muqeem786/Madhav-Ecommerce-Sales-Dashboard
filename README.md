# Madhav Ecommerce Sales Dashboard

## Project Overview

This project presents an interactive Ecommerce Sales Dashboard developed in Power BI to analyze sales performance, profitability, customer behavior, and product category trends. The dashboard enables business users to monitor key performance indicators and identify opportunities for growth.

## Dashboard Preview

![Dashboard Screenshot](Screenshot%202025-02-25%20194137.png)

## Key Metrics

* Total Sales Amount: 438K
* Total Profit: 37K
* Total Quantity Sold: 5615
* Average Order Value (AOV): 121K

## Features

* Quarterly Sales Analysis
* State-wise Sales Performance
* Category-wise Quantity Analysis
* Payment Mode Analysis
* Customer-wise Sales Analysis
* Monthly Profit Trends
* Sub-Category Profit Analysis
* Interactive Filters and Slicers

## Tools Used

* Power BI
* Power Query
* DAX
* Excel

## Key Insights

* Maharashtra generated the highest sales revenue.
* Clothing accounted for the largest share of product quantity sold.
* COD was the most preferred payment method.
* Printers and Bookcases generated the highest profit among sub-categories.
* Profit showed strong performance during January and November.
